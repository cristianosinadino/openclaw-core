# Validation Failure Escalation

| Trigger | Action | Contact |
| --- | --- | --- |
| Validation gate fails (tests, review, QA) | Stop execution, document failure in the task log, request guidance | REPLACE_ME (initiative owner) |
| Safety/compliance rule conflict | Halt work, summarize conflict, await explicit override | REPLACE_ME (security lead) |
| Cross-initiative impact detected | Freeze related changes, notify responsible initiative owner, align on mitigation plan | REPLACE_ME (program manager) |
| Blocker > 24h without response | Escalate with summary + proposed options | REPLACE_ME (primary sponsor) |

All escalations must cite the relevant requirement/task ID and link to evidence/logs.

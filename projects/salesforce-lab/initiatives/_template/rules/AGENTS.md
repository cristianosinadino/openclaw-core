# AGENTS

Initiative-local operating rules for AI and human contributors.

## Mandatory Startup Sequence

1. Read this file first.
2. Read `../README.md` (initiative execution contract).
3. Read required documents in `../docs/` for the current task.
4. Start only from items in `../requirements/active/`.

## Execution Guardrails

- Work only inside this initiative unless explicitly requested.
- Maintain traceability from each change to a requirement/story.
- Prefer minimal, testable increments over broad refactors.
- Update docs when behavior or architecture changes.

## Conflict Handling

- Follow rule precedence from initiative `README.md`.
- If guidance is unclear or conflicting, stop and ask for clarification.
